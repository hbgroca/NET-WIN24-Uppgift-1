Några frågor? Hitta svaren i FAQs nedan

Har du några obesvarade frågor eller vill komma i kontakt med oss?

Lagrar appen någon av mina personuppgifter?

Din säkerhet är vår högsta prioritet. Vi lagrar endast den information som krävs för att tillhandahålla våra tjänster, och all data krypteras för att skydda din integritet.

Vilka format kan jag ladda ner min transaktionshistorik i?

Du kan ladda ner din transaktionshistorik i flera format, inklusive PDF och Excel, så att du kan hantera dina data på det sätt som passar dig bäst.

Kan jag schemalägga framtida överföringar?

Ja, du kan enkelt schemalägga framtida överföringar i appen. Välj datum och belopp, så sköter appen resten!

När kan jag använda tjänsterna i Bankappen?

Tjänsterna i appen är tillgängliga dygnet runt, så du kan hålla koll på din ekonomi och göra transaktioner när som helst, var som helst.

Kan jag skapa ett eget lösenord som är lätt för mig att komma ihåg?

Absolut! När du registrerar dig kan du välja ett lösenord som är säkert men samtidigt lätt för dig att komma ihåg.

Vad händer om jag glömmer eller tappar bort mitt lösenord?

Om du glömmer eller tappar bort ditt lösenord kan du enkelt återställa det genom att följa stegen i appen. Du kommer att få ett e-postmeddelande med instruktioner för att skapa ett nytt lösenord.
